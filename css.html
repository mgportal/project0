CSS.

CSS stands for Cascade Style Sheets.

Applying CSS.

  • Attribute style. 
    <h1 style="color:blue;text-align:center;">Heading</h1>
  • STYLE tag in HEAD section. 
    <style>
    h1 {
      color: blue;
      text-align: center;
    }
</style>
  • CSS file (.css). LINK tag in HEAD section. 
    <link rel="stylesheet" href="styles.css">

CSS selectors.

#name Identifies <TAG id="name">
.name Identifies <TAG class="name">
a, b Multiple element selector
a b Descendant selector. Format when inside, at whatever level
a > b Child selector. Format when direct child
a + b Abjacent Sibling selector
[a=b] Attribute selector. (input[type=text])
a:b Pseudoclass selector. (button:hover)
a::b Pseudo element selector. (a::before, a::after, p::selection)

Responsive Design.

Viewport is an HTML5 feature that refers to user's visible area of a web page.
We need to include this line in all your pages: 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

CSS @media rule is used in media queries to apply different styles for different media types/devices.

Flexbox is a CSS module that defines a CSS box model optimized for user interface design.

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

Bootstrap is a framework for responsive design. It is based in Flexbox.
To use it we just need to copy-paste link from bootstrap site.
