Git.

Git is a Version Control Tool.
The most important characteristics of git are:

  1. Keep track of changes to code.
  2. Sycronize code between different people.
  3. Test changes without affecting the original.
  4. Revert back to previous versions of code.
Git reference.

GitHub.

GitHub is a web site that stores git repositories.
Each repository is identified by an URL.

GitHub Pages.

GitHub publish repositories on the web so that can be accessed and tested from anywhere.

Git basic commands.

  • git clone url. Download a copy of the repository locally.
  • git add file. Marks a local file to be included in the next commit. It is sent to the staging area.
  • git commit -m "message". Snapshot of the local repository including new changes. Files must be added previouly.
  • git status. Shows the working tree status.
  • git push. Uploads local repository to GitHub.
  • git pull. Downloads changes from GitHub to local repository.
  • git log. Shows a list of all commits. Each commit is identified by a hash string.
  • git reset --hard hash-string. Reverts repository to the specified commit.
  • git reset --hard origin/master. Reverts repository to GitHub original version.
  • git reflog. Manages reference log information.

Branching.

  • git branch. Lists branches.
  • git branch mybranch. Creates a branch called mybranch.
  • git checkout mybranch. Changes current branch.
  • git commit -am "message". Adds all changed files and commits.
  • git merge mybranch. From master brach, merges branch master with mybranch.
  • git push --set-upstream origin mybranch. Uploads to GitHub including mybranch.

Remotes.

  • git fetch. Download commits from GitHub when local repository is behind remote.
  • git merge origin/master. Points HEAD to last changes.
  • git pull. Combines git fetch and git merge origin/master.

Forks and Pull Requests.

A Fork is a copy of a repository completely separate from the original.
A Pull Request allows you tell others about your changes.

Software requirements.

  • Create an account on GitHub.
  • Download and install Git.
