Python.

Python is an interpreted programming language.
We can execute our programs with $python file.py or use $python to open console.
The Python Language Reference.

Flask.

Flask is a microframework written in Python. Manages HTTP requests and responses.
To run the Flask web server we use $flask run.
An environment variable must be set with $export FLASK_APP=application.py.
Flask uses Jinjer2 template language.
